Arduino Basics Course
Course Topics :
1. Introduction to Arduino
2. An Overview of C & C++
3. Controlling the input and output of the control system
4. Simulation Using Protues.
5. DC Motor control
6. Stepper Motor
7. Analog Sensor (Temperature sensor)
8. LCD
9. PWM
10. Servo motor
11. UART
12. Ultrasonic
13. Project of first level
NOTE: this section contain 5 practice lecture, 2 theoretical Lecture and 4 Simulation.
The objective of this Section is:
• Make you familiar with the better HW used to day to Control Your Robot or any smart applications.
• On this course you will understand Arduino Kit and most of input output types.
• Know how to make a lot of programs with the C language.
• Learning HW connections and make some projects.
• After this course you will be ready to be embedded developer.
